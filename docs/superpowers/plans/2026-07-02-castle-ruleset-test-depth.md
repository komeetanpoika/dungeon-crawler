# Castle Ruleset + Depth-6 Test Level Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Derive a complete `castle` ruleset from the user's castle-demo painting and render it on a new cheat-only depth-6 test level, proving the generator can fully skin a playable map.

**Architecture:** A committed Node script (`tools/derive-castle-ruleset.mjs`) tags the painting's tiles and runs the editor's pure `deriveRules()` into a new `rulesets.json` key. Data additions to `levels.js` wire depth 6 to the new ruleset; `parseLevelCheat` validates against `LEVEL_CONFIG` instead of `FINAL_DEPTH`. The game's existing `decorateMap` does the rendering — no game-engine changes.

**Tech Stack:** Vanilla JS (ESM), `node:test`, `playwright-core` (`_electron`) for runtime verification.

**Spec:** `docs/superpowers/specs/2026-07-02-castle-ruleset-test-depth-design.md`

## Global Constraints

- Do NOT modify: `LEVEL_CONFIG` entries for depths 0–5, `DEPTH_THEMES` entries for depths 0–5, the `catacombs`/`outdoors`/`moss` rulesets, `renderer/data/painter-maps.json`, or `FINAL_DEPTH` (stays 5).
- The derived ruleset is a NEW `rulesets.json` key: `castle`.
- `renderer/data/rulesets.json` currently has 2-space indent and NO trailing newline — the script's write must match exactly (`JSON.stringify(obj, null, 2)`, no `+ '\n'`), or editor saves will produce diff noise.
- Tests are `node:test` files in `test/`, run via `npm test`. Baseline: 354 passing.
- Runtime checks must leave `renderer/data/` clean except the intended `rulesets.json` change (verify with `git status --short renderer/data/`).

```js
// Derive the 'castle' ruleset from the castle-demo painting using the editor's
// pure deriveRules. Re-run after repainting the source map:
//   node tools/derive-castle-ruleset.mjs
// Fails without writing if any painted tile is missing from TABLE (skipped > 0)
// or any tagged tile has no sprite on disk.
import { readFileSync, writeFileSync, existsSync } from 'node:fs'
import { dirname, join } from 'node:path'
import { fileURLToPath } from 'node:url'
import { deriveRules } from './tile-editor/derive-rules.js'

const ROOT = join(dirname(fileURLToPath(import.meta.url)), '..')
const MAPS_FILE = join(ROOT, 'renderer/data/painter-maps.json')
const RULESETS_FILE = join(ROOT, 'renderer/data/rulesets.json')
const SOURCE_RULESET = 'catacombs'                    // where the painting is stored
const SOURCE_MAP = 'castle-demo-1781607145194'
const TARGET = 'castle'

// tag, role, member tiles. Only tiles that appear exclusively on the overlay
// layer may take the overlay role; dual-layer tiles keep their base role
// (deriveRules then counts their overlay appearances as "no overlay").
const TABLE = [
  ['castle.floor', 'floor', [
    'tile_0048', 'tile_0050', 'tile_0030', 'tile_0042']],
  ['castle.wall', 'wall', [
    'tile_0000', 'tile_0002', 'tile_0004', 'tile_0005', 'tile_0006',
    'tile_0012', 'tile_0013', 'tile_0015', 'tile_0016', 'tile_0017',
    'tile_0018', 'tile_0026', 'tile_0028', 'tile_0045', 'tile_0057',
    'tile_0059']],
  ['overlay.castle', 'overlay', [
    'tile_0001', 'tile_0014', 'tile_0019', 'tile_0031', 'tile_0064',
    'tile_0065', 'tile_0066']],
]

const tileMeta = new Map()
for (const [tag, role, names] of TABLE)
  for (const name of names) tileMeta.set(name, { role, tags: [tag] })

for (const name of tileMeta.keys()) {
  if (!existsSync(join(ROOT, 'renderer/assets/tiles', `${name}.png`))) {
    console.error(`missing sprite: ${name}.png — aborting`)
    process.exit(1)
  }
}

const store = JSON.parse(readFileSync(MAPS_FILE, 'utf8'))
const map = store[SOURCE_RULESET]?.maps?.[SOURCE_MAP]
if (!map) {
  console.error(`map not found: ${SOURCE_RULESET}/${SOURCE_MAP} — aborting`)
  process.exit(1)
}

const frag = deriveRules(map.base, map.overlay, tileMeta)
if (frag.skipped > 0) {
  console.error(`${frag.skipped} painted cells not covered by TABLE — aborting`)
  process.exit(1)
}

const rulesets = JSON.parse(readFileSync(RULESETS_FILE, 'utf8'))
rulesets[TARGET] = { tiles: frag.tiles, tags: frag.tags }
// Match the editor's save format exactly: 2-space indent, no trailing newline.
writeFileSync(RULESETS_FILE, JSON.stringify(rulesets, null, 2))
console.log(`wrote ruleset '${TARGET}': ${Object.keys(frag.tiles).length} tiles, ` +
  `${Object.keys(frag.tags).length} tags`)
```

import { LEVEL_CONFIG } from '../data/levels.js'

// Returns a valid starting depth when `buffer` ends with "level<N>"
// (N must have a LEVEL_CONFIG entry — currently 0..6), otherwise null.
// Matches on the suffix so stray earlier keystrokes don't block a later
// valid code. Case-insensitive.
export function parseLevelCheat(buffer) {
  const m = /level(\d+)$/.exec(String(buffer).toLowerCase())
  if (!m) return null
  const depth = Number(m[1])
  return LEVEL_CONFIG.some(c => c.depth === depth) ? depth : null
}
```

// Render the depth-6 castle level in the real game and screenshot it.
const { _electron } = require('playwright-core')
const path = require('path')
const SHOT_DIR = process.env.SCRATCH || __dirname

;(async () => {
  const app = await _electron.launch({
    args: ['.'],                        // game mode (no --editor)
    cwd: '/home/lappemikb/projects/dungeon-crawler',
    env: { ...process.env, DISPLAY: ':0' },
  })
  const win = await app.firstWindow()
  const consoleErrors = []
  let decorateFallbacks = 0
  win.on('console', (m) => {
    if (m.type() === 'error') consoleErrors.push(m.text())
    if (m.type() === 'warning' && m.text().includes('decorate:')) decorateFallbacks++
  })
  win.on('pageerror', (e) => consoleErrors.push(`pageerror: ${e.message}`))

  // Title menu, then type the cheat.
  await win.waitForSelector('#menu-overlay .menu-panel', { timeout: 15000 })
  await win.waitForTimeout(500)
  await win.keyboard.type('level6', { delay: 60 })
  await win.waitForTimeout(2500)        // level gen + sprite load + first frames

  const menuGone = await win.evaluate(() =>
    document.getElementById('menu-overlay').style.display === 'none' ||
    !document.querySelector('#menu-overlay .menu-panel'))
  console.log('CHEAT_STARTED_LEVEL', menuGone ? 'PASS' : 'FAIL')

  await win.screenshot({ path: path.join(SHOT_DIR, 'castle-level-initial.png') })

  // Walk a few tiles in two directions to prove the level is playable.
  for (const key of ['ArrowRight', 'ArrowRight', 'ArrowRight', 'ArrowDown', 'ArrowDown']) {
    await win.keyboard.down(key)
    await win.waitForTimeout(350)
    await win.keyboard.up(key)
    await win.waitForTimeout(120)
  }
  await win.screenshot({ path: path.join(SHOT_DIR, 'castle-level-after-walk.png') })

  console.log('CONSOLE_ERRORS', consoleErrors.length, consoleErrors.length === 0 ? 'PASS' : 'FAIL')
  if (consoleErrors.length) console.log(consoleErrors.slice(0, 5).join('\n'))
  console.log('DECORATE_FALLBACK_WARNINGS', decorateFallbacks)

  await app.close()
})().catch((e) => { console.error('VERIFY_FAIL', e); process.exit(1) })
```
