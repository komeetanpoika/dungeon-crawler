# Game Menu System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a title menu (on launch), a pause menu (Esc during play), and a game-over screen, plus an "Open Editor" entry — replacing the boot-straight-into-play and "press R to restart" behavior.

**Architecture:** A small pure `phase.js` state machine (title/playing/paused/gameover) gates the game loop's `update()`. A DOM overlay module `ui/menu.js` renders the three screens and takes callbacks. `game.js` holds the phase, wires menu callbacks to transitions, and shows the title on `init()`. New IPC opens the editor / quits the app.

**Tech Stack:** Vanilla ES modules, Electron renderer, Canvas 2D + DOM overlay, `node --test` runner.

## Global Constraints

- Test runner: `npm test` (full) or `node --test test/<file>.js`. ES modules (`"type": "module"`).
- "Base game" plays exactly as today; the title menu's Play button is laid out so a future "Custom" play button can be added beside it (base-vs-custom asset mode is OUT OF SCOPE).
- The `requestAnimationFrame` loop runs in every phase; it only calls `update(delta)` while phase is `playing`, and only calls `render()` when a run exists (`state` is non-null).
- Menu navigation: clickable buttons AND keyboard (↑/↓ move selection, Enter activates, Esc pauses/resumes).
- Title stat line format: `Deepest: Level {deepestReached} · Runs: {runsCompleted} · Treasure: {✓|✗}` (✓ when `treasureStolen`).
- `renderer/game.js`, `renderer/ui/menu.js`, and DOM/Electron code are NOT imported by `node --test` (the project's established testing boundary). They are verified via `node --check` + a runtime Electron boot. Only the pure pieces (`phase.js`, `menu.formatMetaSummary`) are unit-tested.
- `menu.js` must not touch `document` at module top level (only inside functions), so `node --test` can import `formatMetaSummary`.

```javascript
// Game phase state machine. Pure: no DOM, no game state.
export const PHASE = {
  TITLE: 'title',
  PLAYING: 'playing',
  PAUSED: 'paused',
  GAMEOVER: 'gameover',
}

const ALLOWED = {
  title:    ['playing'],
  playing:  ['paused', 'gameover'],
  paused:   ['playing', 'title'],
  gameover: ['playing', 'title'],
}

export function canTransition(from, to) {
  return (ALLOWED[from] ?? []).includes(to)
}
```

```javascript
// Overlay menu screens (title / pause / game over). DOM-only; receives callbacks.
// Keep all document access inside functions so the pure helper stays importable
// under node --test.

let keyHandler = null
let currentButtons = []
let selectedIndex = 0

function overlayEl() { return document.getElementById('menu-overlay') }

export function formatMetaSummary(meta) {
  const treasure = meta.treasureStolen ? '✓' : '✗'
  return `Deepest: Level ${meta.deepestReached} · Runs: ${meta.runsCompleted} · Treasure: ${treasure}`
}

function clearKeyHandler() {
  if (keyHandler) { window.removeEventListener('keydown', keyHandler); keyHandler = null }
}

function highlight() {
  currentButtons.forEach((b, i) => b.classList.toggle('selected', i === selectedIndex))
}

function renderScreen({ title, subtitle, buttons }) {
  const el = overlayEl()
  el.innerHTML = ''

  const panel = document.createElement('div')
  panel.className = 'menu-panel'

  const h = document.createElement('h1')
  h.className = 'menu-title'
  h.textContent = title
  panel.appendChild(h)

  if (subtitle) {
    const s = document.createElement('div')
    s.className = 'menu-subtitle'
    s.textContent = subtitle
    panel.appendChild(s)
  }

  currentButtons = buttons.map(({ label, onSelect }) => {
    const btn = document.createElement('button')
    btn.className = 'menu-btn'
    btn.textContent = label
    btn.addEventListener('click', () => onSelect())
    panel.appendChild(btn)
    return btn
  })

  el.appendChild(panel)
  el.style.display = 'flex'
  selectedIndex = 0
  highlight()

  clearKeyHandler()
  keyHandler = (e) => {
    if (e.key === 'ArrowDown') {
      selectedIndex = (selectedIndex + 1) % buttons.length; highlight(); e.preventDefault()
    } else if (e.key === 'ArrowUp') {
      selectedIndex = (selectedIndex - 1 + buttons.length) % buttons.length; highlight(); e.preventDefault()
    } else if (e.key === 'Enter') {
      buttons[selectedIndex].onSelect(); e.preventDefault()
    }
  }
  window.addEventListener('keydown', keyHandler)
}

export function showTitle(meta, { onPlay, onOpenEditor, onQuit }) {
  renderScreen({
    title: 'DUNGEON CRAWLER',
    subtitle: formatMetaSummary(meta),
    buttons: [
      { label: 'Play', onSelect: onPlay },
      { label: 'Open Editor', onSelect: onOpenEditor },
      { label: 'Quit', onSelect: onQuit },
    ],
  })
}

export function showPause({ onResume, onRestart, onQuitToTitle }) {
  renderScreen({
    title: 'Paused',
    buttons: [
      { label: 'Resume', onSelect: onResume },
      { label: 'Restart', onSelect: onRestart },
      { label: 'Quit to Title', onSelect: onQuitToTitle },
    ],
  })
}

export function showGameOver({ won, deepestLevel }, { onPlayAgain, onQuitToTitle }) {
  renderScreen({
    title: won ? '🏆 Victory!' : '💀 You Died',
    subtitle: `Reached Level ${deepestLevel}`,
    buttons: [
      { label: 'Play Again', onSelect: onPlayAgain },
      { label: 'Quit to Title', onSelect: onQuitToTitle },
    ],
  })
}

export function hide() {
  clearKeyHandler()
  const el = overlayEl()
  el.style.display = 'none'
  el.innerHTML = ''
}
```
